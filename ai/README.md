# ai/

Machine-learning artifacts and pipelines for Ather.

- `datasets/` — labeled training data
- `embeddings/` — precomputed embedding stores
- `recommendation-models/` — feed/explore ranking models
- `moderation-models/` — toxicity, spam, NSFW classifiers
- `llm-prompts/` — prompt templates for AI assistant
- `vector-memory/` — long-term agent memory stores
- `training/` — training scripts and configs
- `evaluation/` — eval harnesses and metrics
- `notebooks/` — exploratory notebooks
- `inference/` — inference servers and adapters

Scaffold — populate as ML work begins.
