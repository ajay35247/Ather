# infra/terraform/

Terraform infrastructure-as-code. Scaffold — modules to be added.

- `aws/` — root AWS account/provider wiring
- `networking/` — VPC, subnets, NAT, security groups
- `eks/` — EKS cluster, node groups
- `rds/` — Postgres RDS instances
- `redis/` — ElastiCache Redis
- `kafka/` — MSK / self-managed Kafka
- `s3/` — buckets for media / backups
